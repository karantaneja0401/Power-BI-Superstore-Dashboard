This project presents an interactive Power BI dashboard developed to analyze and monitor overall business performance for a retail superstore.
The dashboard is designed with a strong focus on business KPIs, growth analysis, and clear storytelling to support quick and effective decision making.

## Key Highlights
- Total Revenue, Revenue YTD, YoY Growth & Profit Margin KPIs  
- Month and Year based interactive slicers  
- Category & Sub Category contribution analysis  
- Region wise revenue comparison  
- YTD vs Last Year trend analysis using DAX  

## Tools Used
- Power BI
- DAX (YTD, YoY, CALCULATE)
- Data Modeling
- Data Visualization
